JSX Transformer Service

This is a service to transform JSX to JS, via POST requests. Currently this simply uses the react-tools npm package. This uses v0.10 of that package. There may be inconsistencies if you are using a different version of React.

Usage

There are 3 available params, file, code, and harmony. file is the file that should be transformed. code is the raw string of JSX (not to be used in conjunction with file). harmony enables ES6 features of the transformer.

A curl request may look like this: 

curl jsx-transformer.herokuapp.com \
     -X POST
     -F "file=@path/to/file" \
     -F "harmony=true"

If you are interested in extending this service, patches welcome!
